Подробное описание
Назначение скрипта:
Комплексная оптимизация Windows 10 через удаление ненужных компонентов и настройку системы с помощью интуитивного консольного интерфейса.

Основные возможности:
Удаление приложений (30+):

Microsoft Store, Cortana, Feedback Hub

Mail и Calendar, Your Phone, Skype

Погода, Новости, Карты

Камера, Запись голоса, Кино и ТВ, Groove Music

Clipchamp, Sticky Notes, To Do, Калькулятор

Paint 3D, Photoshop Express, 3D Viewer

Solitaire Collection, Xbox, Mixed Reality

UWP версии Office

Настройки системы:

Изменение значка поиска в панели задач

Скрытие кнопки Cortana

Скрытие кнопки Просмотра задач

Удаление избранного из панели задач

Ультимативная очистка меню "Пуск":

Полное удаление всех плиток (Edge, Office, развлечения, инструменты)

Очистка кэша CloudStore

Сброс layout меню "Пуск"

Удаление закрепленных элементов

Автоматический перезапуск Explorer

Особенности интерфейса:
Управление стрелками - навигация вверх/вниз по списку

Клавиша ПРОБЕЛ - отметка/снятие выделения элементов

Клавиша ENTER - запуск выполнения выбранных действий

Визуальная индикация - цветовое выделение выбранных элементов

Подробное логирование - запись всех действий в файл

Технические особенности:
Автоматическая проверка прав администратора

Массовое удаление пакетов Appx

Глубокие изменения реестра

ИНСТРУКЦИЯ ПО ПРИМЕНЕНИЮ

Скачайте файлы:

WinConfig.ps1 (основной скрипт)

WinConfig.bat (запускающий файл)

Поместите оба файла в одну папку

Запустите двойным кликом: WinConfig.bat

Подробная инструкция
Способ 1: Простой запуск (рекомендуется)
Нажмите правой кнопкой на файл WinConfig.bat

Выберите "Запуск от имени администратора"

Скрипт автоматически:

Установит нужные политики выполнения

Запустит основной скрипт

Покажет интерактивное меню

Способ 2: Ручной запуск через PowerShell

# Откройте PowerShell как администратор и выполните:
Set-ExecutionPolicy RemoteSigned -Force
.\WinConfig.ps1

Управление в скрипте:

Стрелка ВВЕРХ - перемещение вверх по списку

Стрелка ВНИЗ - перемещение вниз по списку

ПРОБЕЛ - выбрать/снять элемент (отметка [X])

ENTER - запустить выполнение выбранных действий

Принудительная очистка кэша меню "Пуск"

Безопасное завершение процессов
